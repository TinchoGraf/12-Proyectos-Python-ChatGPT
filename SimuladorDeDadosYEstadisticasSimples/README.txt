# 🎲 Simulador de Dados y Estadísticas Simples

Este programa simula el lanzamiento de uno o varios dados de 6 caras y muestra estadísticas de los resultados.

## 🚀 Cómo funciona
1. El usuario ingresa la cantidad de lanzamientos.
2. Se generan valores aleatorios entre 1 y 6.
3. Se muestran:
   - La suma total de los lanzamientos.
   - El promedio de los resultados.
   - El número más repetido.
   - La lista completa de valores obtenidos.

## 🧠 Conceptos aplicados
- Funciones
- Bucles `while`
- Listas
- Librerías `random` y `collections.Counter`
- Manejo de errores con `try/except`

## 🏁 Ejemplo de salida
Ingrese la cantidad de lanzamientos: 5
La suma total de los dados tirados es de: 19
El promedio de los dados tirados es de: 3.8
El valor del dado más repetido fue el siguiente: 4
Los resultados de los dados tirados son los siguientes:
[4, 5, 1, 4, 5]
