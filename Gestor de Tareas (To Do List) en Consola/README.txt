# ✅ Gestor de Tareas (To-Do List) en Consola

Aplicación de consola que permite crear, listar, completar y eliminar tareas, con guardado automático en archivos de texto.

## 🚀 Funcionalidades
- Agregar nuevas tareas pendientes.
- Ver tareas pendientes y completadas.
- Marcar tareas como completadas.
- Eliminar tareas pendientes.
- Guardado automático al salir.

## 🧠 Conceptos aplicados
- Listas y bucles `while`
- Funciones y modularización
- Manejo de excepciones (`try/except`)
- Archivos de texto (`with open`)
- Menús interactivos por consola

## 🏁 Ejemplo de ejecución
--- Gestor de Tareas ---

Agregar Tarea

Ver Tareas

Eliminar Tarea

Completar Tarea

Salir
Seleccione una opción (1-5): 2
